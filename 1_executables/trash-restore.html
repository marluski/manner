NAME

trash-restore - Restore for Command line trash utility.

SYNOPSIS

trash-restore [OPTION] ... [PATH]

DESCRIPTION

Use to restore a trashed file or directory to the original path. This command is a part of the trash-cli package that provides a command line interface trashcan utility compliant with the FreeDesktop.org Trash Specification. It remembers the name, original path, deletion date, and permissions of each trashed file.

PATH

Destination file path to restore from the trash. If omitted, an
interactive

selection interface is used.

OPTIONS

--overwrite

Overwrite existing files with files coming out of the trash.

--sort {date,path,none}

Sort list of restore candidates by given field (default: date).

--trash-dir=TRASHDIR

Restore from TRASHDIR, for files that were trashed using a custom TRASHDIR.

-h, --help

Show help message and exit.

--version

Show the program’s version number and exit.

EXAMPLES

$ trash-restore
0 2007-08-30 12:36:00 /home/andrea/foo
1 2007-08-30 12:39:41 /home/andrea/bar
2 2007-08-30 12:39:41 /home/andrea/bar2
3 2007-08-30 12:39:41 /home/andrea/foo2
4 2007-08-30 12:39:41 /home/andrea/foo
What file to restore [0..4]: 4
$ ls foo
foo
