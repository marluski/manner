NAME

xkbcli-dump-keymap-x11 — Debugger for XKB keyboard keymaps on X11

SYNOPSIS

xkbcli-dump-keymap-x11 [options]

DESCRIPTION

xkbcli-dump-keymap-x11 is a commandline tool to dump XKB keymaps from X servers.

This requires an X server to be running.

This is a debugging tool, its behavior or output is not guaranteed to be stable.

--help

Print help and exit
