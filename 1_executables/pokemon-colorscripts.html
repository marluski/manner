NAME

pokemon-colorscripts - display unicode sprites of pokemon in the terminal

SYNOPSIS

pokemon-colorscripts [-h|--help] [-l|--list] [-r|--random generation] [-n|--name name [-s|--shiny]] [--no-title]]

DESCRIPTION

Pokemon-colorscripts is a CLI utility written in shell script that prints out unicode sprites of pokemon to the terminal. Almost all pokemon generations 1-8 are available (except different forms). You can select specific pokemon to be printed out by name or let the script choose a pokemon randomly. While choosing randomly you can also filter by generation.

OPTIONS

-h, --help

Print help.

-l, --list

List all available pokemon.

-r, --random generation

Select a pokemon at random and print it out. Can be followed by an optional generation number or range 1-8 to print out random pokemon from a specific generation or range of generations. If no generation is specified, selects from full list of available pokemon.

-n, --name name

Select a specific pokemon to print out by name. Most pokemon are spelled as expected. A few pokemon with some special characters in their names might be spelled slightly differently. Examples nidoran-f,nidoran-m,farfetchd,mr-mime etc. You can check the output of the list option if in doubt.

-f, --form form

Display an alternate form of the pokemon

-s, --shiny

Show the shiny version of the pokemon instead

-b, --big

Show a larger size sprite

--no-title

Do not display pokemon name

FILES

All the files are present in the directory

"/usr/local/opt/pokemon-colorscripts"

with the executable script being symlinked to

"/usr/local/bin/pokemon-colorscripts"

EXAMPLES

pokemon-colorscripts -r

Prints out a random pokemon. Add this command to your shell config file (.bashrc, .zshrc, fish.config etc.) to print out a random pokemon every time you open the terminal.

pokemon-colorscripts -r 1

Print out a random pokemon from generation 1.

pokemon-colorscripts -r 1-3

Print out a random pokemon from generations 1-3.

pokemon-colorscripts -r 1,3,6

Print out a random pokemon from generations 1,3 or 6.

pokemon-colorscripts -l

Print out all available pokemon names.

pokemon-colorscripts -n chikorita

Print out the pokemon chikorita.

pokemon-colorscripts -n mudkip -s

Print out the shiny version of mudkip.
