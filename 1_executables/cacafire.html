NAME

cacademo, cacafire - libcaca’s demonstration applications

SYNOPSIS

cacademo

cacafire

DESCRIPTION

This manual page documents briefly the cacademo and cacafire programs.

cacademo displays ASCII art effects with animated transitions: metaballs, moire pattern of concentric circles, old school plasma, Matrix-like scrolling.

cacafire displays burning ASCII art flames.

KEYS

Enter

forces an effect transition to happen

Space

pauses and resumes the program

Esc

exits the program

BUGS
