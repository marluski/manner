NAME

g-ir-generate - Typelib generator

SYNOPSIS

g-ir-generate [OPTION...] FILES...

DESCRIPTION

g-ir-generate is an GIR generator, using the repository API. It generates GIR files from a raw typelib or in a shared library (--shlib). The output will be written to standard output unless the --output is specified.

OPTIONS

--help

Show help options

--shlib=FILENAME

The shared library to read the symbols from.

--output=FILENAME

Save the resulting output in FILENAME.

--version

Show program's version number and exit
