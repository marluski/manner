NAME

zvbi-ntsc-cc - closed caption decoder

SYNOPSIS

zvbi-ntsc-cc [ options ]

DESCRIPTION

zvbi-ntsc-cc reads vbi data from /dev/vbi and decodes the enclosed cc data. Start it with ’-h’ to get a list of cmd line options.
